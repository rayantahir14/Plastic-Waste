Data Context/Cleaning

Context of Data:

The dataset I will be utilizing captures the global audit results from plastic waste that is collected from volunteers. The waste was then used to link pollution back to specific companies and the materials found to better learn how to combat the pollution

Data Cleaning:

The Janitor function was used to clean the column names and standardize them, mutate was used to convert strings into numeric and add a year column. Country names were extracted and stored in a new column for the 2019 data set. In the 2019 data set duplicate columns were removed also inconsistent column names were renamed. Finally the data sets were combined into one singular data set

1. Dominant Plastic Type by Country

find_max <- function(x) {
  names(x)[x == max(x, na.rm = TRUE)][1] 
}

dominant_plastic <- function(data) {
  plastic_cols <- 
    c(
    "empty", 
    "hdpe", 
    "ldpe", 
    "o", 
    "pet", 
    "pp", 
    "ps", 
    "pvc")
  dominant <- apply(
    select(data, all_of(plastic_cols)), 
    1,
    FUN = find_max
  )
  return(dominant)
}

3. Plastic Type Summary by Year

plastic_summary_by_year <- function(data, year_value) { 
data |> 
filter(year == year_value) |> 
summarise( pet = sum(pet, na.rm = TRUE), 
hdpe = sum(hdpe, na.rm = TRUE), 
ldpe = sum(ldpe, na.rm = TRUE), 
pp = sum(pp, na.rm = TRUE), 
ps = sum(ps, na.rm = TRUE), 
pvc = sum(pvc, na.rm = TRUE), 
other = sum(o, na.rm = TRUE) ) |> 
mutate( total = pet + hdpe + ldpe + pp + ps + pvc + other ) |>
mutate( 
pet = pet / total, 
hdpe = hdpe / total, 
ldpe = ldpe / total, 
pp = pp / total, 
ps = ps / total, 
pvc = pvc / total, 
other = other / total, 
year = year_value 
) 
}
